Myself with my dog Hachi I play the MacOs Tiles (or QT) version, which can be found in Nethack.org, configured for ASCII color play. The configuration file is a file called .nethackrc in your home directory, mine looks like: OPTIONS=!autopickup,ascii_map, color, hilite_pet, lit_corridor, showexp, showscore, boulder:0;
